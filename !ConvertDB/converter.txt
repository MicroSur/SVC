	Распространение:
 Программа Convert2SVC распространяется как бесплатный продукт
 и является частью пакета Sur Video Catalog (далее SVC).

	С чего начать:
Откройте базу-источник для импорта совместимого формата (mdb, xml, csv, amc)
 кнопкой [Открыть источник]
Выберите таблицу или ноду с полями о фильмах в поле со списком.
Выберите поле в списке-источнике, проверьте доступ к данным 
 кнопкой [Проверка данных]
Откройте базу формата Sur Video Catalog [Фильмы] или [Актеры]
Выберите соответствующие пары в списках полей открытых баз, добавьте нужные связи 
 кнопкой [Добавить связь].
Для объединения нескольких входящих полей, проделайте ту же операцию, помечая поля 
из списка-источника и уже существующей связи. 
Используйте разделители. По умолчанию - скобки.
Кнопкой [Связать похожие] программа сделает попытку связать поля источника 
 и приемника автоматически.
В завершении, выполните импорт данных в SVC базу. 
Данные SVC базы дополняются новыми, старые данные сохраняются.
Закройте программу импорта! Используйте базу в каталогизаторе SurVideoCatalog. 
Поиск дубликатов (если указано) может занять некоторое время.

Для CSV файлов (текстовый файл с данными, разделенными запятыми):
 Сначала установите правильный разделитель полей, если разделитель не запятая.
 После загрузки файла-источника выберите доступные для SVC базы поля из 
 поля со списком и присвойте их соответствующим колонкам в таблице-источнике,
 кликнув по заголовку поля в таблице.
 Для объединения нескольких полей, присвойте колонкам одинаковые названия.
 Возможно менять порядок полей для объединения перетаскиванием заголовков.
 Для указания поля проверки на дубликаты, кликните на заголовок нужного поля,
  при нажатой галочке [Не импортировать дубликаты]. 
  Делайте это после присвоения названий колонкам в списке.
 Ограничения для csv-файлов:
  Кавычки внутри одного поля недопустимы. 
  Сдвоенные кавычки программа исключит.

Для AMC файлов (Ant Movie Catalog):
 Свяжите Picture c FrontFace.
 Если обложка не находится в самой amc-базе или путь к ней не абсолютный,
 укажите полный путь к папке с обложками для amc-базы. 

Для преобразования XML базы: 
 http://movies.nnov.ru/cat/www.movies.nnov.ru.xml.rar
 убедитесь, что первая строка xml-файла такая:
 <?xml version="1.0" encoding="windows-1251"?>
 Откройте xml-файл, выберите ноду Movie, затем создайте нужные связи.
 Не забудьте связать Picture с CoverPath - в SVC редакторе будет легко 
 получить обложку из Интернет.

	Поля Sur Video Catalog базы, описание и синонимы:

Acter		- Актеры, Actors, Starring
Annotation	- Описание, Description
Audio		- Характеристики звуковых потоков, AudioBitrate, AudioFormat, AudioInfo     
CDN		- Кол-во носителей, Disks, MediaCount
Checked		- Фильм помечен   
Country		- Страна, производитель, Studio   
CoverPath	- URL обложки, Picture (текст) 
Debtor		- Должник, Loaner, Borrower 
Director	- Режиссер
FileLen		- Длина файла, Size, FileSize
FileName	- Имя файла, File(s)
FPS		- Кадров в секунду, FrameRate
FrontFace	- Обложка, Cover, Screen
Genre		- Жанр, Category
Key		- ключевое поле, внешние данные игнорируются
Label		- Метка носителя, Media, MediaLabel, CDBoxID 
Language	- Языки фильма, Languages
MediaType	- Тип носителя
MovieName	- Название фильма, Title, MovieTitle, Name, OriginalTitle, TranslatedTitle
MovieURL	- URL странички фильма, URL
Other		- Примечания, Comments
Rating		- Рейтинг популярности 0-10, Score
Resolution	- Разрешение, формат
SnapShot	- Кадры (1,2,3), Screen
snDisk		- Серийный номер оптического носителя, Serial, CDSerial
SubTitle	- Язык субтитров, SubTitles
Time		- Продолжительность, Length
Video		- Характеристики видео потока, VideoBitrate, VideoFormat, VideoInfo 
Year		- Год выпуска
